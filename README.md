天气预报温度准确率计算
===
本软件通过分析从2009年到2012年对最高最低气温（72小时，48小时，24小时）的预报数据和实际最高最低气温分析，得出气温预报的月、季度准确率。


####环境：
1. scala
2. java

####运行：
在本目录下，windows用户请执行`run.bat`, linux用户请执行`run.sh`，会生成csv形式的报告。

例如，CCTV的2012年季度预报准确率如下：
